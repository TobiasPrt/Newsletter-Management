# Mate - Newsletterverwaltung

Web-Application for organizing Newsletters on a smaller scale.


## Features

- Fully responsive
- Secure Login
- Real-Time Search (Newsletter, Adresses)
- Multiple Changes without multiple http-Requests
- Error Logging
- PHP Mailer embedded
- Quill Editor embedded
